Installation Instructions

For Windows:

Tutorial Video:
  1. Download the plugin at http://sketchuptostl.tk.
  2. Save it somewhere. By default, it is saved to "Downloads". If it asks, choose Downloads.
  3. Open Windows explorer and open the Downloads folder.
  4. Find the file (skp_to_Custom.rb) and copy it.
  5. Go to Computer or This PC.
  6. Open "Local Disc (C:)"
  7. Open the folder "Program Files" or "Program Files x86" if it's there.
  8. Open the folder "Google"
  9. Open the folder "Google SketchUp 8" or similar.
  10. Open the folder "Plugins"
  11. Right click and paste or CNTRL+V.
  12. Press "Continue" or enter the administrator's password.
  13. Open Google SketchUp (Note: The newer versions are just called SketchUp)

For Mac:

Sorry this isn't with screenshots or good instructions beause I have Windows. Please email ben@speedysnail6.info if you have better ones.

  1. Download the plugin at http://sketchuptostl.tk
  2. Save it somewhere. By default, it will save it to downloads, but if it asks just save it to Downloads.
  3. Open Finder.
  4. Click your name in the left sidebar under "Places".
  5. Open the folder "Downloads".
  6. Find the file (skp_to_custom.rb), select it and click Edit > Copy.
  7. Click your name in the left sidebar under "Places".
  8. Open the folder "Library".
  9. Open the folder "Application Support".
  10. Open the folder "Google"
  11. Open the folder "Google SketchUp 8".
  12. Open the folder "Plugins"
  13. Click Edit > Paste.
  14. Open Google SketchIp (Note: The newer versions are just called SketchUp)
  15. Press "Start using SketchUp"
  16. Open or design your 3D model(s).
  17. If you want to export the whole thing:
    1. Make sure that you have nothing selected.
    2. Click on "Tools" in the top of the screen.
    3. Click "Export to STL mm".
    4. Click "Yes".
    5. Choose where to save the STL, and name it.
    6. Make sure to add ".stl" to the end of the file name. (It will probably work anyways, just a good thing to do)
    7. Press "Save".
    8. You're done!
  18. If you want to export only a part of the file:
      1. Select the object(s) to export to a STL by dragging and dropping.
      2. Click on "Tools" in the top of the screen.
      3. Click "Export to STL mm".
      4. Choose where to save the STL, and name it.
      5. Make sure to add ".stl" to the end of the file name. (It will probably work anyways, just a good thing to do)
      6. Press "Save".
      7. You're done!
